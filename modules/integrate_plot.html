integrate_plot.plot_feature_2d(f_post_h5, key='', i1=1, i2=1000000000.0, im=1, iz=0, uselog=1, title_text='', hardcopy=False, cmap=[], clim=[], **kwargs)

Plot a 2D feature from a given HDF5 file.

Parameters:

  • f_post_h5 (str) – Path to the HDF5 file.

  • key (str) – Key of the feature to plot. If not provided, the first key found in the file will be used.

  • i1 (int) – Start index of the feature to plot.

  • i2 (int) – End index of the feature to plot.

  • im (int) – Index of the feature.

  • iz (int) – Index of the z-coordinate.

  • uselog (int) – Flag indicating whether to apply logarithmic scaling to the data.

  • title_text (str) – Additional text to include in the plot title.

  • kwargs – Additional keyword arguments to be passed to the scatter plot.

Returns:

int – 1 if the plot is successful.
